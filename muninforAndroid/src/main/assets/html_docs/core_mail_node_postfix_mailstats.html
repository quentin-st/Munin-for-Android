NAME

postfix_mailstats

postfix_mailstats - Plugin to monitor the number of mails delivered and rejected by postfix

CONFIGURATION

Configuration parameters for @@CONFDIR@@/postfix_mailstats, if you need to override the defaults below:

 [postfix_mailstats]
  env.logdir  - Which logfile to use
  env.logfile - What file to read in logdir

DEFAULT CONFIGURATION

 [postfix_mailstats]
  env.logdir  /var/log
  env.logfile mail.log

AUTHOR
